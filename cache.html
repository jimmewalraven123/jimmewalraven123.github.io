Cache

cache coherentie en vervanging

cache coherentie: Erg belangrijk in multicore-systemen, want zorgt ervoor dat alle CPUs met dezelfde data werken en kan ze ook tegelijk wijzigen.

cache vervanging: Bepaalde algoritmen beslissen welke data worden vervangen wanneer hij vol is, om ervoor te zorgen dat alleen de relevante data bewaard blijft.

Soorten en werking cache

1. Schijfcache: Cache in schijven verbetert de toegangstijd door vaakgebruikte bestanden op te slaan in snel opslaggeheugen, wat heel handig is voor de snelheid van toegang vergeleken met andere harde schijven.

2. Browser en web cache: Verlicht de belastting op het netwerk en de server doordat hij afbeeldingen, scripts en andere dingen lokaal opslaat waardoor paginas sneller laden bij terugkerende bezoeken.

3. CPU-cache: Bestaat uit meerdere niveaus (l1,l2,l3), deze caches slaan veel data en instructies op zodat de CPU deze direct kan gebruiken en daardoor kan de RAM overgeslagen worden, wa veel tijd bespaart.
